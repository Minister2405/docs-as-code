# Quickstart

A simple, step-by-step guide to launching your docs site.

---

## Step 1: Run locally

- Install the Mintlify CLI: `npm i -g mint`
- Open your docs folder in the terminal
- Run: `mint dev`
- Visit `http://localhost:3000`

<Tip>Your preview updates automatically as you edit files.</Tip>

---

## Step 2: Customize your docs

- Open `docs.json`
- Update `name`, `description`, and brand colors
- Add content in `/essentials/overview`

<Note>Keep changes small and preview often.</Note>

---

## Step 3: Publish changes

- Commit your files
- Push to your GitHub repo
- Wait for deployment to finish

<Warning>Don’t mix draft content with published pages in your main branch.</Warning>
